☕ Shah's Cafe – Luxury Cafe Ordering Website

A modern, luxury-inspired cafe ordering website built using HTML, CSS, and JavaScript. This project delivers a premium digital cafe experience with elegant UI design, responsive layouts, interactive animations, and seamless ordering functionality.

🌟 Project Overview

Shah's Cafe is designed as a high-end cafe website where customers can:

Browse premium coffee and food menus
Experience immersive visual effects and animations
Add items to a shopping cart
Place orders through WhatsApp
Send order details via Email
Enjoy a fully responsive mobile-friendly experience

The design focuses on luxury aesthetics with rich colors, smooth transitions, elegant typography, and modern user interactions.
